# Lab Visual Programming - Week 2 - Part 2
Tugas ini untuk membuat sebuah kembang api spesial untuk Ulang Tahun Seseorang :v 
## File dan Struktur Project
- `color.dart`: File ini untuk menyediakan function untuk menghasilkan warna acak, serta mengelola color pallete yang digunakan oleh setiap kembang api untuk menciptakan visual yang menarik (i hope)
- `fireworks.dart`: File ini merupakan kelas utama yang dimana untuk mengontrol fireworks, mengelola posisi, ledakan, dan penggambaran kembang api di terminal
- `main.dart`: File ini untuk menjalankan file fireworks dan color dengan kata lain sebagai fungsi utama. Serta menjalankan loop untuk mengontrol frame terminal dan update dan reset posisi fireworks
